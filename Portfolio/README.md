# Bilal Farooq - Portfolio Project

# CONTENT
My name is Bilal Farooq. This portfolio website is to showcase my education, skills, experience, projects and a little bit about me. My target audience is employers, classmates and alumni. I have experience with C++, Java, Python, HTML, CSS, Javascript as well as tools like VSCode, Jupyter Notebooks and Cursor. I have included a viewable pdf resume (once resume is clicked) that can be dowloaded. I've linked my gmail, linkedin as well as Github accounts for viewers to contact and connect. My desires are to work in something meaningul in the Data Science or Information Technology Industry.

# Design and INTERACTIVITY
The style I chose is not something that I really have seen, I really just combined lots of things I saw online. I used a gradient color scheme for the background which is constantly changing and I've used a bunch of different fonts which range from casual to more creative. For the contrast, since my background is relatively light I decided to go with a darker color to make ti pop more. I've included a navigation bar at the top, followed by an about me section. After this I listed my experiences followed by my projects. In the end I have a footer with my contact section. I've used icons for my experiences (companies I've been at) as well as for my contact information (gmail, linkedIn, Github) which are all clickable. I've implemented a carousel for one of my projects which dispalys screenshots of different parts of the project running.
